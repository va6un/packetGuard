⚡ How to Generate the Input File

1

Capture traffic with Wireshark or tcpdump

Save as a .pcap or .pcapng file.

2

Export using tshark

Run the exact command below in your terminal:

bash 
tshark -r capture.pcap \
    -T fields \
    -e frame.time_relative \
    -e ip.src \
    -e ip.dst \
    -e tcp.srcport \
    -e udp.srcport \
    -e tcp.dstport \
    -e udp.dstport \
    -e _ws.col.Protocol \
    -e frame.len \
    -e tcp.flags.syn \
    -e tcp.flags.ack \
    -e tcp.flags.reset \
    -e eth.src \
    -e eth.dst \
    -E header=y -E separator=, -E quote=d -E occurrence=f \
    > capture.csv
⚠️

Why tshark explicit-field format?

The default Wireshark "Export Packet Dissections" packs multiple values into display-formatted strings. The tshark -T fields format gives one clean value per column with no ambiguity.

🌐

Limitations

IPv4 only. IPv6 packets are detected and reported, but excluded from analysis. Authentication is not required — this is a single-user academic tool.
